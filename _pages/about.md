## 律师简介

**张建国律师**是一名专业的医疗纠纷法律服务提供者，致力于为患者及家属争取合法权益。

### 专业背景

- 执业律师，从事医疗纠纷法律服务多年
- 深入研究医疗损害责任法律制度
- 与多家医疗鉴定机构建立合作关系
- 累计代理医疗纠纷案件数百件

### 服务特色

✅ **专业性**：精通医疗法律法规，了解医学专业知识

✅ **患方立场**：坚定维护患者权益，只代理患方

✅ **灵活收费**：支持风险代理、半风险代理等多种方式

✅ **全程代理**：从咨询、取证、鉴定到诉讼的全方位服务

### 擅长领域

1. **医疗事故责任纠纷**
   - 构成医疗事故的损害赔偿
   - 司法鉴定申请与代理

2. **医疗损害责任纠纷**
   - 医疗过错导致的损害赔偿
   - 知情同意权纠纷

3. **医疗产品责任纠纷**
   - 假药、不合格医疗器械导致的伤害
   - 药物副作用纠纷

4. **鉴定代理服务**
   - 医疗过错司法鉴定
   - 伤残等级鉴定
   - 尸体检验鉴定

### 工作流程

```
初步咨询 → 病历分析 → 正式委托 → 取证调查 → 鉴定申请 → 调解或诉讼 → 案件结案
```
